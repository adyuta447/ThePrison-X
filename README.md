# Install

# For Termux
<pre>
1. apt update && upgrade
2. apt install git
3. git clone https://github.com/alfaridzilinux/deepdark
4. cd deepdark
5. sh deepdark.sh / bash deepdark.sh
</pre>

# For Linux (Debian Based)
<pre>
1. apt-get update 
2. apt-get install git
3. git clone https://github.com/alfaridzilinux/deepdark
4. cd deepdark
5. sudo sh deepdark.sh Or sudo bash deepdark.sh
</pre>
